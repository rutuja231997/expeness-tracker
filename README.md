# SpendWise

SpendWise is a modern React Expense Tracker application that helps users manage and track their daily expenses efficiently.  
The application provides smooth navigation, expense categorization, form validation, and a clean responsive user interface.

---

# Features

- Add new expenses
- Expense categorization
- Form validation using Zod
- Error messages below input fields
- Responsive design using Tailwind CSS
- Context API state management
- Smooth section navigation
- Modal-based expense form
- React Router navigation
- Reusable components architecture
- Clean modern UI

# Installation & Setup

## 1. Clone Repository

```bash
git clone <your-repository-url>
````

---

## 2. Navigate to Project Folder

```bash
cd SpendWise
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Start Development Server

```bash
npm run dev
```

Application will run on:

```bash
http://localhost:5173
```

---
